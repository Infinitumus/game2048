# game2048

Играйте в "2048"! Загрузите JAR-файл и запустите его с помощью Java версии 16+. <br>
Объединяйте плитки с числами, передвигая их. Достигните 2048, чтобы выиграть!

Play "2048"! Download the JAR file and run it with Java 16+.<br>
Combine numbered tiles by sliding them. Reach 2048 to win! <br><br>
